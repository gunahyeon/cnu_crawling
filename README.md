# cnu_crawling
 충남대학교 학생생활관 크롤링(개인 프로젝트)
 
 충남대 학생생활관에 거주하고 있는 학생 및 거주를 고민하고 있는 학생들을 위한 카카오톡 챗봇 개발 중에 있습니다. 
 기존 대학 웹사이트 사용의 복잡성 개선 및 불편함을 해소하고자 기획 및 제작하게 되었습니다.
